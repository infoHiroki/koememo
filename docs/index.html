会議の記録作業を
AIの力で効率化

KoeMemo（コエメモ）は、音声や動画ファイルから文字起こしを行い、LLM（大規模言語モデル）と連携して構造化された議事録を自動生成するツールです。Faster Whisperによる高品質な文字起こしとLLM APIを活用し、会議の記録作業を効率化します。

特徴

全自動処理

指定フォルダを監視し、新しいファイルを自動検出して処理します。一度設定すれば後は自動的に議事録が生成されます。

高品質文字起こし

Faster Whisperモデルによる正確な文字起こしで、複数話者の会話も高精度で認識します。

AI議事録生成

OpenAI、Anthropic、Googleなど各種LLMを活用し、構造化された議事録を自動作成します。

セグメント表示

文字起こし結果をタイムスタンプ付きセグメント単位で表示します。特定の発言を素早く見つけられます。

最小限の設定

シンプルなGUIで必要な設定のみを管理。設定は直感的で使いやすいインターフェースで行えます。

バックグラウンド実行

ユーザーの作業を妨げないサイレント動作。他の作業をしながら議事録を生成できます。

セットアップサポートが必要な場合は有償にて行っています。お問い合わせは info.hirokitakamura@gmail.com までご連絡ください。

Windows向けセットアップ

Windows環境でのインストール手順の概要です。詳細はWindows詳細セットアップガイドを参照してください。

必要条件

  • Python 3.9以上（3.11.3推奨）
  • FFmpeg
  • インターネット接続

インストール手順

  1. 開発者モードを有効化する（必須）
  2. Pythonをインストール（Add to PATHにチェック）
  3. KoeMemoをダウンロード
  4. FFmpegをインストールして環境変数に追加
  5. 必要なPythonライブラリをインストール
詳細な手順を見る

使い方

1

初期設定

KoeMemoを起動し、基本設定タブで各ディレクトリを設定します。LLM API設定タブでAPIキーとモデルを設定します。

2

サービスの開始

設定が完了したら「サービス開始」ボタンをクリックします。KoeMemoはバックグラウンドで動作を開始します。

3

ファイルの配置

設定した入力ディレクトリに音声ファイル（mp3, wav, m4a, flac）または動画ファイル（mp4, avi, mov, wmv）を追加します。

4

自動処理

KoeMemoが自動的にファイルを検出し、文字起こしとAI議事録の生成を開始します。処理の進捗はログビューワータブで確認できます。

5

結果の確認

処理が完了すると、文字起こし結果と議事録が設定したディレクトリに保存されます。
